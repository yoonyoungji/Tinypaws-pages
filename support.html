  • Drag to reposition on the desktop

Permissions

TinyPaws may guide you to macOS Accessibility or Input Monitoring settings so the character can react to keyboard input while other apps are active.

The app is still usable without these permissions, but some keyboard-reactive behavior may be limited.

Compatibility

  • Platform: macOS
  • Minimum version: macOS 13.0 or later
  • App type: menu bar desktop companion

FAQ

The character is not responding to keyboard input.

Check whether Accessibility or Input Monitoring permission has been granted in macOS System Settings.

The character is covering part of my screen.

Drag the character to another position, or adjust the size and transparency from the menu bar controls.

I want to report a bug or request a feature.

Send your macOS version, the TinyPaws version, and a short description of the issue to the support email.

Can I send questions in Korean?

Yes. Korean support is available. If you include your macOS version, TinyPaws version, and reproduction steps, it will help us review the issue faster.
